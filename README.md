## Workstation Upgrade and Configuration

This project involved upgrading and configuring a 15-year-old desktop workstation in order to improve system performance, stop CPU overheating and create a functional environment for technical learning and daily use.

The upgrade was performed while retaining the original hardware platform:

- Intel Core i3-2120 processor
- Gigabyte H61M-S1 motherboard
- Chieftec GPA-350S 350W power supply

Scope of work included:

- hardware compatibility and bottleneck analysis
- RAM upgrade to 16 GB DDR3 Kingston memory
- installation of a dedicated NVIDIA Quadro K2000 graphics card
- multi-monitor workstation setup
- Medion case replacement and cable reorganization
- SSD installation (Goodram CX400) and operating system migration through disk cloning
- configuration of an additional Seagate Barracuda 500 GB HDD for backup and file storage
- replacement of thermal paste on the CPU
- implementation of a USB 2.0 to USB 3.0 front panel adapter due to motherboard limitations
- airflow and cooling optimization analysis

The project required independent technical research, hardware compatibility verification, troubleshooting, and working within the limitations of legacy hardware and a low-budget environment.

As a result, the workstation achieved significantly improved responsiveness, longevity and usability, providing a stable multi-monitor setup suitable for further IT learning and technical projects.
